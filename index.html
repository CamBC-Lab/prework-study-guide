HTML

  • The body elemnt represents the visible content show to the user or the metadata.
  • The H1 element is used for sorting the elements on a webpage onto a hierarchy ranging from H1 to H6
  • The li element is used for unsorted list and can be used for organizing a webpage
  • The ul element is used for sorted list and also is used for organization usually parenting an li element
