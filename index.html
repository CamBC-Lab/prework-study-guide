Prework Study Guide

HTML

  • The head element contains information about the webpage and sets its parameters.
  • The body elemnt represents the visible content show to the user or the metadata.
  • The H1 element is used for sorting the elements on a webpage onto a hierarchy ranging from H1 to H6.
  • The li element is used for unsorted list and can be used for organizing a webpage.
  • The ul element is used for sorted list and also is used for organization usually parenting an li element.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much psace we want around the content inside an element.
  • Inline CSS is used for directly styling specific elements of a webpage in HTML.
  • An Internal style sheet is used to dictate rules for the whole page.
  • An external CSS style sheet is the common method of styling a webpage. It allows storage of all changes on a sperate file.

Git

  • git status: checks the current branch the user is on.
  • git checkout -b brance-name: creates a branch and switches to it.
  • git add -A: commits all files to the working branch.
  • git pull origin main: Initates a pull request on base branch.
  • git commit -m: commits the changes to the repository while adding a message for clarification.
  • cd command: used for moving into different files the current file holds.
  • pwd command: checks the current users file location.
